# Enterprise-Ai-Solutions-Architect

## Week 1 Summary

#### Goal : Understand the fundamentals of Enterprise AI and how AI solutions are designed in real-world enterprise applications.

### Day 1 – Enterprise AI Fundamentals
    Introduction to Enterprise AI
    Difference between consumer AI and Enterprise AI
    Role of an Enterprise AI Solutions Architect

    Key Takeaway: Enterprise AI solves business problems by integrating AI with enterprise systems.

### Day 2 – LLMs, AI Agents & Human-in-the-Loop
    LLM vs AI Agent
    Salesforce as the source of truth
    Human-in-the-Loop (HITL)

Key Takeaway:

    LLM = Thinks
    AI Agent = Thinks + Acts

### Day 3 – Tool Calling & MCP
    Tool Calling
    APIs
    Model Context Protocol (MCP)

Key Takeaway: AI Agents use tools to access enterprise systems, while MCP provides a standard way for AI to communicate with those tools.

### Day 4 – Retrieval-Augmented Generation (RAG)
    What is RAG?
    RAG vs Fine-tuning
    Retrieving enterprise data before generating answers

Key Takeaway: Retrieve first, Generate second. AI answers should be grounded in trusted enterprise data.

### Day 5 – Prompt Engineering
    System Prompt
    User Prompt
    Context Prompt
    Prompt Templates
    Guardrails
    Structured Outputs

Key Takeaway: Well-designed prompts and guardrails ensure AI responses are consistent, safe, and enterprise-ready.
